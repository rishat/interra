 InTerra Blog Machine

- Системные Требования ---

   1. PHP 4.3.0+
   2. MySQL 3.2+
   3. Apache Mod Rewrite
   4. Терпение и чуточку мозгов

- Установка --------------

   1. Качаем дистрибутив
   2. Заливаем его на сервер
   3. Ставим chmod 777 на папки common и templates_c
      Note: Может статься, что надо будет проставить chmod 777 на папку куда устанавливается 
            блог, подобная проблема присутствует, например, на Мастерхосте
   4. Открываем в броузере InTerra и следуем инструкциям на экране
   5. Теоретически всё будет работать
      Note: А если же не будет, то возможно кто-то уже топтался на ваших граблях и это было
            обсуждено на форуме -- http://forum.inses.ru/
